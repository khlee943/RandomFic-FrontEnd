# RandomFic-FrontEnd

__Tech Stack__

+ Next.js (w/ Typescript)
+ Docker to containerize

__Deployment__

For development:

+ Clone repository
+ Run npm i
+ Configure .env file
+ Run npm run dev
+ Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

For production:

+ Same as development instructions

Hosted on Netlify.
